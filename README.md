# Access Pass System Analysis Case

## Problem
Сейчас заявки на пропуск обрабатываются вручную, нет прозрачности статусов и централизованного процесса.

## Goal
Автоматизировать подачу, согласование и обработку заявок.

## Roles
- Employee
- Manager
- Pass Administrator

## Scope
- создание заявки
- согласование
- изменение статуса
- уведомления
